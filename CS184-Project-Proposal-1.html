CS184 Project Proposal: Iridescent Materials

Bethany Lu, Jose Alfaro, Michael Chou, Aashika Jhawar

 

Summary

For the final project we want to extend project 3-2 to support textured models. Specifically, we are interested in iridescent rendering and the implementation of interference colors as a spectral representation of light.

 

Problem Description

Our project Pathtracer doesn’t support textures! Using our previous Pathtracer project as a skeleton, we want to extend functionality so that it is able to support iridescence textures. Iridescence is necessary for the rendering of certain objects such as soap bubbles, butterfly wings, and the surface of CDs to name a few. To handle iridescence, we must be more nuanced in the way we treat light as a wavelength, by having to track phase shifts and refraction along iridescent surfaces. To solve this, we plan on looking into using a wavelength based BRDF function to render thin films and pearlescent surfaces. We can derive the reflectance using Beckmann’s distribution solution, an approach we previously implemented to render Microfacet models in Pathtracer 3-2.

In order to test the accuracy and performance of our implementation we will test our rendering across multiple images that depict iridescent lighting (soap bubbles/butterfly wings/peacock feathers). This will allow us to compare our renderings to original images to determine if our project is working properly. Additionally, by adjusting the sampling rate and multibounce light effects at varying levels, we plan to test the efficacy of our solution as we know that increasing these metrics should yield better images with proper implementation. Furthermore, we intend to update our model to handle different wavelengths of light for each color and will use different input levels for these parameters to test our models accuracy as well.

What do you plan to deliver? describe what you believe you must accomplish to have a successful project and achieve the grade you expect (i.e. your baseline plan -- planning for some unexpected problems would make sense).

Our baseline plan is to extend our pathtracer model to support iridescent materials and project this to a dae model that we have already seen in class.

A problem that we foresee is how to implement light as a wave instead of just a vector, which was our previous approach in Pathtracer. This is necessary in order to render iridescent material because iridescence is an optimal phenomenon caused by the interference between waves of light.

What do you hope to deliver? describe what you hope to achieve if things go well and you get ahead of schedule (your aspirational plan)

We hope to deliver a more advanced rendering system that is able to depict complex material modeling of thin-film materials across various thicknesses. This will allow us to render images in which light reflects and refracts to create iridescent colors on both existing images that we have seen in previous projects as well as new images.
